Parágrafos e quebras de linha 😄

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>

❤

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>

Vamos acrescentar alguns símbolos:

  para Espaço em branco

® para Marca Registrada

© para Compyright

™ para Trade Mark

€ para Euro

£ para Libra

¥ para Yen

¢ para Cent

∅ para Vazio

∑ para Soma

Δ para Delta

< para Menor quê

> para Maior quê

← para seta esquerda

↑ para seta acima

→ para seta direita

↓ para seta abaixo

Vamos acrescentar Emojis

Para pesquisar os códigos de emojis a serem inseridos na página visite o site: emojipedia.org

Para inserir o emoji deve-se verificar o codepoints da figura e no lugar do "U+"" substituir por "&#x"
Lembrando que a letra "x" deve estar minúscula.
